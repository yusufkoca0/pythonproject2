How to run: python3 assignment3.py input.txt

Alternative chess game:

Changed rules:
1- Pawns can only move 1 square. First move of 2 squares is removed.
2- No checks and check mates, game is over when a king is captured.
3- Horses can move 1 square diagonally.

Commands:

initialize: defaults the board.

showmoves x: x is any piece. It shows the all possible moves of x.

move x y: x is any piece, y is any square. If x can go to square y, it moves the piece else it prompts an error.

print: prints the current board.

exit: exits the program.
